# KRE-1

KRE-1 is the Knowledge Research Engine for the Lantern Platform. It provides a governed, source-agnostic knowledge ingestion, normalization, provenance, and retrieval layer for ETS, SignalForge, OpsHelm, Christina, and future Lantern services.

## Sprint

Current sprint: **KRE-001 — Enterprise Knowledge Foundation**

## Initial scope

- Canonical knowledge document and chunk schemas
- Connector abstraction
- Markdown and GitHub ingestion
- Normalization and chunking pipeline
- Provider-neutral embedding abstraction
- Search API contract
- Provenance-first metadata
- CI, tests, and architecture documentation

Repository bootstrap in progress.
